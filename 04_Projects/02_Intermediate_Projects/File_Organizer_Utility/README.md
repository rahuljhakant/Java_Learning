# File Organizer Utility

A comprehensive file organization utility that demonstrates intermediate Java programming concepts including file system operations, Java NIO, and practical utility development.

## Features

- Organize files by type (Documents, Images, Videos, Audio, etc.)
- Organize files by date (creation/modification date)
- Organize files by size (Small, Medium, Large, Very Large)
- Organize files by extension
- Clean duplicate files
- Organize by custom criteria (name patterns, content type, permissions)
- Directory analysis and statistics
- File system operations and management

## Learning Objectives

- Implement file system operations using Java NIO
- Use streams and collections for file processing
- Handle file operations safely and efficiently
- Build practical utility applications
- Apply file management best practices
- Use advanced Java features for file handling
- Implement comprehensive file organization logic

## How to Run

1. Compile the Java file:
   ```bash
   javac FileOrganizer.java
   ```

2. Run the program:
   ```bash
   java FileOrganizer
   ```

## Usage

1. **Organize by File Type**: Automatically categorize files into folders based on their type
2. **Organize by Date**: Group files by their creation or modification date
3. **Organize by Size**: Categorize files based on their size ranges
4. **Organize by Extension**: Group files by their file extensions
5. **Clean Duplicate Files**: Find and remove duplicate files
6. **Custom Organization**: Organize files by custom criteria
7. **Directory Analysis**: Analyze directory structure and file statistics

## Features

- **File Type Detection**: Automatically detect file types based on extensions
- **Size Categorization**: Categorize files by size ranges
- **Duplicate Detection**: Find duplicate files using hash comparison
- **Safe Operations**: Prevent overwriting existing files
- **Comprehensive Analysis**: Detailed directory and file statistics
- **User-Friendly Interface**: Interactive menu-driven interface
- **Error Handling**: Robust error handling for file operations

## File Organization Categories

- **Images**: JPG, PNG, GIF, BMP, SVG
- **Videos**: MP4, AVI, MKV, MOV, WMV, FLV
- **Audio**: MP3, WAV, FLAC, AAC, OGG
- **Documents**: PDF, DOC, DOCX, TXT, RTF
- **Archives**: ZIP, RAR, 7Z, TAR, GZ
- **Executables**: EXE, MSI, DEB, RPM
- **Other**: Files that don't match any category
