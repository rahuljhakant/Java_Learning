## ğŸ”§ Development Setup

### Prerequisites
- Java 17 or higher
- Maven 3.6+ (for advanced projects)
- IntelliJ IDEA, Eclipse, or VS Code
- Git (for version control)

### Setup Instructions
1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd Java_Tutorial/04_Projects
   ```

2. **Choose a project**
   ```bash
   cd 01_Beginner_Projects/Calculator
   ```

3. **Compile and run**
   ```bash
   javac *.java
   java Calculator
   ```

4. **For Maven projects**
   ```bash
   mvn clean compile
   mvn exec:java -Dexec.mainClass="com.tutorial.projects.advanced.UserController"
   ```
