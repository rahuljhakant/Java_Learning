# Simple Banking System

A comprehensive banking system application that demonstrates intermediate Java programming concepts including collections, user input handling, and object-oriented design.

## Features

- Account creation and management
- Deposit and withdrawal operations
- Balance inquiry
- Money transfer between accounts
- Account information display
- Menu-driven user interface
- Input validation and error handling

## Learning Objectives

- Implement a complete banking system
- Use collections (HashMap) for data management
- Handle user input and create interactive menus
- Apply object-oriented programming principles
- Implement proper error handling and validation
- Build a real-world application with multiple classes

## How to Run

1. Compile the Java files:
   ```bash
   javac *.java
   ```

2. Run the program:
   ```bash
   java BankingSystem
   ```

## Code Structure

- `Account.java` - Account class with banking operations
- `BankingSystem.java` - Main application with user interface

## Usage

1. Create accounts with account numbers, names, and initial balances
2. Perform deposits and withdrawals
3. Check account balances
4. Transfer money between accounts
5. View all accounts in the system
