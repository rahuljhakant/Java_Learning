# Calculator Project

A simple calculator application that demonstrates basic Java programming concepts including classes, methods, and user input handling.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- User-friendly interface
- Error handling for division by zero
- Input validation

## Learning Objectives

- Create and use classes and objects
- Implement methods with parameters and return types
- Handle user input and output
- Implement basic error handling
- Apply object-oriented programming principles

## How to Run

1. Compile the Java file:
   ```bash
   javac Calculator.java
   ```

2. Run the program:
   ```bash
   java Calculator
   ```

## Code Structure

- `Calculator.java` - Main calculator class with all operations
