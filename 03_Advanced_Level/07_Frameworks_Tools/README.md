### Build Tools
- **Maven**: XML-based project management
- **Gradle**: Groovy/Kotlin-based build system
- **Dependencies**: External library management
- **Build Lifecycle**: Compile, test, package, deploy

### Testing Framework
- **JUnit**: Unit testing framework
- **Test Cases**: Individual test methods
- **Assertions**: Test result validation
- **Mocking**: Test isolation and stubbing

### Application Framework
- **Spring Boot**: Rapid application development
- **Auto-configuration**: Automatic bean configuration
- **REST APIs**: Web service development
- **Dependency Injection**: Inversion of control

### ORM Framework
- **Hibernate**: Object-relational mapping
- **Entity Classes**: Java objects mapped to database tables
- **HQL**: Hibernate Query Language
- **Session Management**: Database session handling
