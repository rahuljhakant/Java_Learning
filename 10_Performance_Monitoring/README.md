## Performance Optimization Principles

- **Measure First**: Always measure before optimizing
- **Profile Don't Guess**: Use profiling tools to identify bottlenecks
- **Optimize Hot Paths**: Focus on frequently executed code
- **Consider Trade-offs**: Balance performance with maintainability
- **Test Changes**: Validate performance improvements with testing
